# NMhelper Privacy Site

Static site for **https://privacy.nmhelper.in** (Play Store privacy policy URL).

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main privacy policy (styled) |
| `delete-data.html` | Data deletion requests (Play Store Data safety URL) |
| `privacy-policy.html` | Redirects to `index.html` (legacy URLs) |
| `logo192.png` | Brand icon (copy from `NmClient-main/public`) |
| `CNAME` | `privacy.nmhelper.in` for GitHub Pages |

## Deploy

Push to GitHub Pages branch; ensure custom domain `privacy.nmhelper.in` points to Pages.

## Update logo

Replace `logo192.png` when the app logo changes.
